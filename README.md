### 家政服务管理小程序✔Java✔SSM✔Vue2✔Uniapp✔MySQL✔万字文档
## 【项目采用技术】
①架构: B/S、MVC
②系统环境：Windows/Mac
③开发环境：IDEA、JDK1.8、Maven、MySQL、Tomcat7.5-9、HBuilderX/微信开发者工具
④技术栈：Java、MySQL、Tomcat7.5-9、Spring、SpringMVC、Mybatis/MybatisPlus、Vue2、Uniapp、ElementUI、HTML5（H5）、JavaScript、CSS、图片上传
【系统功能模块】
本系统包含管理员、家政阿姨、用户3个角色。
①管理员角色：
服务项目管理：管理员可以增删改查服务项目信息。
留言板管理：管理员可以回复用户的留言。
家政阿姨管理：管理员可以增删改查家政阿姨信息，并查看家政阿姨的详细信息。
家政公司管理：管理员可以增删改查家政公司信息，并查看家政公司的详细信息。
②家政阿姨角色：
家政公司：家政阿姨可以查询家政公司信息，并查看家政公司的详细信息。
家政预约：家政阿姨可以查看用户提交的家政预约信息，并审核家政预约信息。
留言板：家政阿姨可以发布留言，其留言内容需要管理员查看并回复。
③用户角色：
家政公司：用户可以查看家政公司介绍信息，并预约家政服务。
家政预约：用户可以查看家政预约信息，并支付家政预约订单，同时可以评价家政预约信息。
评价信息：用户可以查看评价详细信息。
